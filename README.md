# Kalendar — веб-планер морских судов

Приложение для автоматического построения графика захода судов на полугодие с браузерным интерфейсом.

## Основные возможности

- выбор судна и создание циклического маршрута по портам;
- автоматический расчет дат прихода/отхода на 182 дня;
- заглавная страница показывает расписание в табличном виде:
  - колонки по портам,
  - подколонки `Приход`/`Отход`,
  - формат удобен для визуального контроля;
- ручное изменение дат прямо в таблице и кнопка **«Обновить расписание»**,
  которая автоматически сдвигает последующие заходы;
- техническая страница:
  - матрица переходов между каждым портом (в сутках),
  - базовая стоянка отдельно для каждого порта (в сутках),
  - сохранение параметров в `planner_db.json`.

## Запуск

```bash
python3 planner.py --host 0.0.0.0 --port 8000
```

Открыть в браузере:

```text
http://localhost:8000
```

## Хранение данных

Все данные сохраняются в `planner_db.json`:

- матрица переходов между портами;
- базовая стоянка в портах;
- созданные планы.
