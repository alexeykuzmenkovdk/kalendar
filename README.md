# Kalendar — планер морских судов

Программа для автоматического построения графика захода судов на полугодие.

## Что изменилось

Теперь у проекта есть **графический интерфейс (Tkinter)** и **экспорт расписания в табличный вид**, похожий на форму в Word:

- первая строка — заголовок с судном и периодом;
- вторая строка — названия портов (каждый порт занимает 2 колонки);
- третья строка — подзаголовки `Приход` / `Отход`;
- далее строки циклов маршрута, пустые значения заполняются символом `¤`.

Экспорт выполняется сразу в два файла:
- `*.html` — удобно открыть в Word/браузере и распечатать;
- `*.csv` — для редактирования в Excel.

## Возможности

- выбор судна:
  - т/х «Анатолий Иванов»
  - т/х «Ерофей Хабаров»
  - т/х «Русский Восток»
  - т/х «Механик Красковский»
- добавление портов в маршрут и циклическое повторение на 182 дня;
- автоматический расчет дат прибытия/отхода;
- изменение даты прибытия с автоматическим сдвигом последующих заходов;
- ручная правка прибытия/отхода;
- технический раздел:
  - длительность перехода между портами (сутки),
  - длительность стоянки в порту (сутки).

## Запуск

### GUI (рекомендуется)

```bash
python3 planner.py
```

### CLI (консольный режим)

```bash
python3 planner.py --cli
```

При первом запуске создается `planner_db.json` с исходными данными.

## Логика расчета

- горизонт планирования: 182 дня;
- маршрут повторяется по кругу;
- прибытие = отход из прошлого порта + переход;
- отход = прибытие + стоянка.
